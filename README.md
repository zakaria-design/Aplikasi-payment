
# Aplikasi Payment ( Zeky Payment )

## Deskripsi

Aplikasi Payment adalah aplikasi web sederhana yang dibuat menggunakan HTML, CSS, JavaScript, dan Bootstrap 5. Aplikasi ini digunakan untuk mensimulasikan proses pembayaran dengan tampilan yang responsif dan mudah digunakan. dengan memberikan fitur pembayaran sekolah, listrik, pembelian token listrik, pulsa pakety data, pembayaraan multi finance seperti ( fif dan womfinance ).
aplikasi ini juga mendukung pembayaran bpjs dengan fitur multi payment memudahkan pengguna menentukan metode pembayaran 

---

## Teknologi yang Digunakan

- HTML5
- CSS3
- JavaScript
- Bootstrap 5 (CDN)
- pdf deploy by CDN

## Cara Menjalankan

1. Download atau clone repository ini.
2. Buka folder project.
3. Jalankan dan klik kanan file `index.html`
4. pilih open with live server ( wajib install extensi live server pada vscode jika belum ada )

## Fitur

- Halaman utama (Landing Page)
- Halaman pembayaran ( listrik, pdam dan lain-lain )
- Validasi input
- Perhitungan total pembayaran
- Tampilan responsif
- cetak struk
- edit biodata 
- pembelian pulsa
- scan qr code
- metode pembayaran yang fleksibel
